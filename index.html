Rackmaker

Network/Server Rackdrawing program using C++ and Latex to produce PDF files.

View project on GitHub

Welcome to RackMaker.

RackMaker is designed to make the lives of network engineers, server engineers, system engineers, network technicians, server technicians and interns of all walks easier. Ditch the Visio, Auto-Cad or other propriety file type for rack drawing creation and enjoy the beauty of LaTeX generated PDF's.

Best Practice

This software is best used on an Ubuntu 16.04.02 running Appache2, gcc version 5.4.0, and texlive-full. It is not intended for use on evil windows machines.

Creating pages manually

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually.
